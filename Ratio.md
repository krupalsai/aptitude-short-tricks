# ⚖️ Ratio & Proportion — Short Tricks

## Basics

- Ratio a:b means a/b
- Proportion: a:b = c:d → a×d = b×c (cross multiply)

## Key Tricks

### Trick 1: Dividing in a Ratio
Total ÷ (sum of parts) × each part

> Example: Divide 200 in ratio 3:5
> Sum = 8 → First = (3/8)×200 = **75**, Second = (5/8)×200 = **125**

### Trick 2: If A:B = 2:3 and B:C = 4:5, find A:C
```
A:B = 2:3    →   ×4  →  8:12
B:C = 4:5    →   ×3  → 12:15
∴ A:C = 8:15
```

### Trick 3: Compounded Ratio
A:B:C when A:B = 2:3 and B:C = 3:5 → A:B:C = **2:3:5**

### Trick 4: Mean Proportional
Mean proportional of a and b = √(a×b)
> √(4×16) = √64 = **8**

## Practice Problems

1. Ratio of 3:4, total = 84. Find each part. → *21 and 63*
2. A:B = 5:7, B:C = 3:5. Find A:B:C → *15:21:35*
