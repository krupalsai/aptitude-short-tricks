# % Percentages — Short Tricks

## Core Formula

```
Percentage = (Part / Whole) × 100
```

## Fraction ↔ Percentage Table (Memorize!)

| Fraction | Percentage |
|----------|-----------|
| 1/2 | 50% |
| 1/3 | 33.33% |
| 1/4 | 25% |
| 1/5 | 20% |
| 1/6 | 16.67% |
| 1/7 | 14.28% |
| 1/8 | 12.5% |

## Key Shortcuts

### Trick 1: x% of y = y% of x
- 8% of 50 = 50% of 8 = **4** ✅ (much easier!)

### Trick 2: Successive % Change
- Two increases of a% and b%: `net = a + b + (ab/100)`
- Example: 10% then 20% increase → 10 + 20 + 2 = **32%**

### Trick 3: % Increase → Original
- If value increased by 20% to reach 120, original = 120 / 1.2 = **100**

## Practice Problems

1. 15% of 240 = ?  → *Answer: 36*
2. What % is 45 of 180? → *Answer: 25%*
3. A price rises 10% then 10%. Net change? → *Answer: +21%*
