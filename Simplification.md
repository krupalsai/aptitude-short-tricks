# 🔢 Simplification — Short Tricks

## BODMAS Rule

```
B - Brackets  ( ), { }, [ ]
O - Orders    (powers, roots)
D - Division  ÷
M - Multiplication ×
A - Addition  +
S - Subtraction -
```

## Squaring Tricks

### Numbers ending in 5:
```
45² → 4×5 | 25 = 2025
75² → 7×8 | 25 = 5625
```
> Rule: (n5)² = n×(n+1) followed by 25

### Near 100:
```
97² = (97-3)×100 + 3² = 9400 + 9 = 9409
```

## Multiplication Shortcuts

### Multiply by 11:
```
36 × 11 = 3_(3+6)_6 = 396
47 × 11 = 4_(4+7)_7 = 4_(11)_7 = 517
```

### Multiply by 99:
```
56 × 99 = 56 × 100 - 56 = 5600 - 56 = 5544
```

## Approximation Tips

- Round to nearest 10/100 for estimation
- 4.98 × 201 ≈ 5 × 200 = 1000 (actual: 1000.98)

## Practice Problems
