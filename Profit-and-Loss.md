# 💰 Profit and Loss — Short Tricks

## Key Formulas

```
Profit       = SP - CP
Loss         = CP - SP
Profit %     = (Profit / CP) × 100
Loss %       = (Loss / CP) × 100
SP           = CP × (100 + P%) / 100
CP           = SP × 100 / (100 + P%)
```

## Shortcut Table

| Situation | Formula |
|-----------|---------|
| Gain x% | SP = CP × (100+x)/100 |
| Loss x% | SP = CP × (100-x)/100 |
| Find CP from SP with gain% | CP = SP × 100/(100+x) |
| Find CP from SP with loss% | CP = SP × 100/(100-x) |

## Clever Tricks

### Trick 1: Two articles at same SP, one 20% profit, one 20% loss
```
Net result = ALWAYS a LOSS of (x²/100)%
= 20²/100 = 400/100 = 4% loss
```

### Trick 2: Dishonest Shopkeeper (false weights)
```
Profit% = (True weight - False weight) / False weight × 100
```

### Trick 3: Marked Price & Discount
```
Discount% = (MP - SP) / MP × 100
SP = MP × (100 - Discount%) / 100
```
